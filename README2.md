# AI Legal Compliance 
A full-stack prototype designed to bridge the gap between regulatory requirements (EU AI Act, UAE AI Ethics) and technical implementation.

## 🛠️ Tech Stack
- **Backend:** Python (FastAPI)
- **Database:** SQLite
- **Frontend:** JavaScript, HTML5, CSS3

## ⚖️ Key Features
- **Regulatory Logic:** Automated risk scoring based on the 4-tier framework of the EU AI Act.
- **Dynamic Dashboard:** A searchable frontend that filters risk alerts in real-time.
- **Data Integrity:** Includes a test suite to ensure database accuracy and compliance logging.

## 📁 Project Structure
- `main.py`: The API hub.
- `dashboard.html`: The visual interface for compliance officers.
